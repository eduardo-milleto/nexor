# Nexor

Nexor is an integrated enterprise platform that combines ERP, CRM, Finance, Inventory, Email, and AI-powered tools into a unified business management solution.

## Overview

This platform provides a comprehensive suite of applications designed to streamline business operations. The modular architecture allows organizations to adopt components incrementally based on their needs.

## Core Modules

**ERP Dashboard** - Centralized enterprise resource planning with real-time analytics and reporting.

**CRM** - Customer relationship management with contact tracking, pipeline management, and communication history.

**Finance** - Financial management including invoicing, expense tracking, and reporting.

**Inventory** - Stock management, warehouse tracking, and supply chain visibility.

**Email** - Integrated email client with organization-wide communication tools.

**Jupiter AI** - AI-powered assistant for data analysis, automation, and intelligent recommendations.

**Admin Panel** - User management, permissions, and system configuration.

## Technology Stack

- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Radix UI for accessible components
- React Router for navigation

## Requirements

- Node.js 18.x or higher
- npm 9.x or higher

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/eduardo-milleto/nexor.git
cd nexor/frontend
npm install
```

## Development

Start the development server:

```bash
cd frontend
npm run dev
```

The application will be available at http://localhost:5173

## Build

Create a production build:

```bash
cd frontend
npm run build
```
